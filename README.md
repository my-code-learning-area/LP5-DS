     sudo apt-get remove openjdk*
```

     sudo apt update
```

     sudo apt install openjdk-8-jdk openjdk-8-jre

# Install Apache Netbeans (It will be Easier to work with Netbeans on Windows than Ubuntu)

    sudo bash [file_name.sh]
